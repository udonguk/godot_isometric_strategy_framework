# 제품 요구사항 정의서 (PRD)

## 1. 개요 (Overview)
이 문서는 "Vampire Spread Isometric" 게임의 핵심 기능과 요구사항을 정의합니다.

## 2. 핵심 기능 리스트 (Core Features)

### 2.1. 타일 및 맵 시스템 (Tile & Map)
- [ ] **그리드 시스템**: 아이소메트릭 타일맵 생성 및 관리
- [ ] **타일 상태 관리**: 일반 / 감염 중 / 감염 완료 상태 구분

### 2.2. 게임플레이 (Gameplay)
- [ ] **전파(Spread) 로직**: 
    - 인접 타일(Neighbor) 감지 알고리즘
    - 포위 면 수(1~4면)에 따른 점령 속도 가중치 적용
- [ ] **플레이어 조작**: 전파를 주도하는 캐릭터 이동 또는 커서 조작
- [ ] **스노우볼링(Game Loop)**: 점령지 비율에 따른 글로벌 버프/가속

### 2.3. UI/UX
- [ ] **HUD**: 체력, 스킬 쿨타임 표시
- [ ] **메뉴**: 타이틀 화면, 일시정지, 게임 오버

## 3. 우선순위 (Priorities)
- **P0 (필수 구현)**: 캐릭터 이동, 기본 공격, 전파 시스템 기초
- **P1 (중요)**: 적 AI, 레벨 디자인
- **P2 (추가 기능)**: 고급 스킬, 상점, 다양한 적 타입
